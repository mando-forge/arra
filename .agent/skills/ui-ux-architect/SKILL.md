# Skill: UI/UX & Theme Architect

## When to use this skill
Use this during the `Build` phase when implementing front-end interfaces, landing pages, or dashboards.

## How to use it
- **Composition over Creation:** Combine `@aliimam` registry blocks with standard shadcn form elements to create high-end layouts. 
- **Beautiful Defaults:** Ensure padding, margins, and typography scale perfectly according to the `radix-lyra` theme.
- **Dark Mode:** Ensure all components seamlessly utilize the CSS variables defined in the Tailwind v4 index for perfect light/dark mode transitions.
- **No Slop:** Reject overcrowded UIs. Enforce whitespace, clear visual hierarchy, and accessible contrast ratios.
